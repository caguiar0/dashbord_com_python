# 📊 Análise de Vendas de Supermercado com Dashboard Interativo

## 🚀 Visão Geral

Este projeto tem como objetivo analisar dados de vendas de um supermercado e apresentar insights de negócio por meio de um dashboard interativo desenvolvido em Python.

O foco da análise é responder perguntas estratégicas como:

* Qual o faturamento ao longo do tempo?
* Quais produtos geram mais receita?
* Qual filial performa melhor?
* Qual forma de pagamento é mais utilizada?
* Como estão as avaliações dos clientes?

---

## 🛠️ Tecnologias Utilizadas

* **Python**
* **Pandas** → Manipulação de dados
* **Streamlit** → Criação do dashboard interativo
* **Plotly Express** → Visualizações gráficas

## 📊 Dataset

O dataset utilizado contém informações de vendas de um supermercado, incluindo:

* Data da compra
* Cidade (filial)
* Tipo de produto
* Forma de pagamento
* Valor total da compra
* Avaliação do cliente

---

## 📈 Funcionalidades do Dashboard

O dashboard permite visualizar:

### 📅 Filtro por mês

* Seleção dinâmica de período

### 📊 Faturamento por dia

* Evolução das vendas ao longo do tempo

### 🛍️ Faturamento por tipo de produto

* Identificação das categorias mais lucrativas

### 🏪 Faturamento por filial

* Comparação de desempenho entre cidades

### 💳 Faturamento por forma de pagamento

* Distribuição de métodos de pagamento

### ⭐ Avaliação dos clientes

* Média de satisfação por filial
